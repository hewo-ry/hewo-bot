# Hewo-bot

Helper bot of Herwannan Vapaaohjelmoijat Ry.

## Current features

### Google Calendar Sync

Sync Google Calendars to Discord Events. Sync happens every hour.

#### Add Calendar to server

Adds given Google Calendar to Server.

```bash
/calendar add <google-calendar-id> <channel-id>
```

Channel ID currently not used. In the future changes can be logged into that channel.

#### Manually run calendar update

Manually runs update.

```bash
/calendar update
```
